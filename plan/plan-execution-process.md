# Plan execution

1. go through the existing plan and plan the list of actions that needs to be taken to execute each step which is not marked completed.
2. Ask for confirmation to execute the plan or make some amends to list of actions, Execute the step only after receiving confirmation.
3. Once the plan is executed update the readme.md file
4. Update the plan file and mark the actions that have been completed.
4. bump the minor version - bun run version:minor
