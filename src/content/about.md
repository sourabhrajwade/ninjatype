# <i class="material-icons">info</i> About NinjaType

NinjaType is a minimalist typing interface designed to help you improve your typing speed and accuracy. Built with modern web technologies, it provides a clean, distraction-free environment for practicing your typing skills.

## Features

- **Minimalist Design**: Focus on what matters - your typing
- **Real-time Statistics**: Track your words per minute (WPM) and accuracy
- **Multiple Themes**: Choose from a variety of beautiful themes to suit your preference
- **Customizable Settings**: Adjust word count, time limits, and other parameters
- **Keyboard Visualization**: See your progress in real-time with an interactive keyboard display

## Technology Stack

NinjaType is built using:
- **Astro** - Fast, content-focused web framework
- **React** - Interactive UI components
- **TypeScript** - Type-safe development
- **Cloudflare Pages** - Fast, global deployment
